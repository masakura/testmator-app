# Bootstrap アプリ自動操作ツール

Bootstrap アプリを自動操作するツールの実験です。

- PageObject 規定型
- Bootstrap アニメーション完了の待機
- メソッドチェーンによる UI 操作

[GitHub Page](http://masakura.github.io/testmator-app/) で実際に動かせます。Test#1 ボタンをクリックしてください。未圧縮のテストコードもどきは [test.js](app/scripts/test.js) にあります。

テストコードがイマイチ読みやすくならない... jQuery でちまちまやるよりはましだけど。
